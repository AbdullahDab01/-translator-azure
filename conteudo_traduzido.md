# Tradutor do Azure

## Por que usar

- O Azure Translator fornece tradução neural de alta qualidade para múltiplos idiomas.
- A API REST é simples de integrar em aplicações Python e pipelines de documentação.

### Conceitos-chave

- Recurso de tradução: expõe a API REST com autenticação por chave.
- Idiomas suportados: verifique a lista atualizada na documentação oficial do Azure.

> A tradução automática acelera a leitura de artigos técnicos sem perder terminologia.
